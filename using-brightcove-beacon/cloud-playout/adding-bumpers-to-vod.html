はじめに

チャンネルでサーバーサイド広告（SSAI）を有効にしていない場合でも、バンパーを含めることをお勧めします。これは、社内広告やその他の種類の発表やプロモーションである可能性のあるロール前またはロール後の短い動画です。バンパーを使用して、ライブプログラミングセグメント用のバッファーを作成することもできます。

最初のチャネル作成で行う必要のあることは何もありません-オプションバンパーを挿入チャネルが作成された後、自動的に表示されます。これは、ライブストリームまたはSSAIが有効になっているチャネルを含むすべてのチャネルに適用されます

同じビデオに複数のバンパーを追加したり、ビデオと同じようにプログラム順にバンパーを並べ替えたりすることができます。ただし、これを行うには注意が必要です。そうしないと、EPGが不正確になる可能性があります。見るEPGとバンパー詳細については。

バンパーはいいえに表示されますEPGただし、バンパーの持続時間は、ビデオの前後に追加されます（プレロールかポストロールかによって異なります）。

バンパーコンテンツの有効化

バンパーはアカウント内のどの動画でもかまいませんが、の一つ次のタグ：

  • cp-プレロール-バンパー
  • cp-postroll-バンパー
潜在的な問題：
  • これらのタグの両方を同じビデオに追加した場合でも、バンパーとして機能しますが、 EPG不正確になります。
  • 同様に、SSAI対応チャネルの場合、広告プレースホルダーはポストロール広告として扱われます。プレロールバンパーの後に広告プレースホルダーを配置しないでください。これにより、プレロールが効果的に作成されます。EPGが不正確になるポストロールシーケンス。

バンパーの挿入

  1. バンパーを挿入する前に、チャンネルに動画やライブイベントを追加すると保存する変更。
  2. 1つまたは複数のビデオやライブイベントを選択して、バンパーを挿入。[1-1]
  3. の中にバンパーダイアログ選択プレロールまたポストロール次にクリックします入れる挿入したいバンパービデオの横。[1-2]
  4. これで、バンパーがプログラムリストに表示されます。

注

  • [1-1] 複数のアイテムを選択すると、同じバンパーがそれぞれに追加されます。アイテムごとに異なるバンパーが必要な場合は、1つずつ追加します。
  • [1-2] されているビデオのみを覚えておいてくださいバンパーとしてタグ付けリストに表示されます。

バンパーの移動または取り外し

バンパーをドラッグアンドドロップして、プログラムリストの新しい場所に移動します。

バンパーを取り外すには、バンパーを選択してクリックします消去、をクリックします消去再度確認ダイアログで。
